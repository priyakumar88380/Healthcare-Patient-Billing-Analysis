# Healthcare Patient & Billing Analysis

## Project Overview
This project provides a 360-degree view of hospital operations by merging patient clinical data with financial billing records. I used **Power BI** for data cleaning and **Tableau** to build an interactive dashboard.

## Key Sheets
* **Admissions by Condition:** Highlighting top diagnoses.
* **Billing Heatmap:** Visualizing cost concentrations.
* **Patient Demographics:** Age and gender distribution.
* **Hospital Trends:** KPI tracking over time.

## Files in this Repo
* `Tableau Packaged Workbook.twbx`: The interactive dashboard.
* `My first Tableu project.pdf`: A static export of the final visuals.
